Remove or hide titles from posts and pages in WordPress from your website.


== Description ==


**Title Hide** is a FREE, lightweight and user-friendly WordPress plugin that gives you control over the visibility of titles for pages and single posts on your website. Whether you need to show or hide titles, Title Hide has got you covered.


== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/title-hide/` directory, or install the plugin directly through the WordPress plugins screen.
2. Activate the plugin through the '**Plugins**' menu in WordPress.


== Usage ==
1. Activate the plugin through the '**Plugins**' menu in WordPress.
2. Create a new post/page or edit an existing one.
3. On the edit screen, you'll find a checkbox labeled "**Hide title**". Check this box to hide the title.
4. Publish or update the page/post to apply the changes.
